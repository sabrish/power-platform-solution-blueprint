# Power Platform Solution Blueprint (PPSB)

Generate comprehensive technical documentation for Dataverse and Power Platform solutions with automated discovery, analysis, and export capabilities.

## Documentation

For complete documentation, installation instructions, and usage guide, please visit:

**[Main Repository Documentation](https://github.com/sabrish/power-platform-solution-blueprint)**

## Quick Start

This tool runs inside [PPTB Desktop](https://powerplatformtoolbox.com).

1. Install PPTB Desktop
2. Load tool from this directory's `index.html`
3. Connect to your Dataverse environment
4. Select scope (solutions or publishers)
5. Generate blueprint

## Features

- **Multi-scope analysis**: By solution, publisher, or all unmanaged components
- **Comprehensive discovery**: Entities, plugins, flows, security, web resources, and more
- **Interactive UI**: Searchable lists, detailed views, ERD visualization, architecture diagrams
- **Multiple export formats**: Markdown, HTML, JSON, ZIP package
- **Security analysis**: Roles, profiles, field security, attribute masking
